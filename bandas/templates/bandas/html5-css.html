Seletores CSS

Os seletores CSS são usados para selecionar elementos HTML com base em seus atributos, classes, IDs, etc. Abaixo está uma tabela que documenta os seletores que usamos nesta aplicação.

Tipo de Seletor Usado Comentário
Classe Usado para estilizar a .header, .footer e .main-content.
ID Usado para estilizar o #nav, #sidebar e #footer.
Atributo Usado para estilizar input[type="text"], a[target="_blank"] e img[alt].
Pseudo-classe Usado para estilizar diferentes estados de links como a:link, a:visited, a:hover e a:active.
Pseudo-elemento Usado para estilizar p::first-line, p::first-letter e a::before.

Combinadores de Seletores

Os combinadores de seletores são usados para selecionar elementos HTML com base em sua relação com outros elementos. Abaixo está uma tabela que documenta os combinadores que usamos nesta aplicação.

Combinador Usado Comentário
Descendente Usado para estilizar .main-content p.
Filho Usado para estilizar .main-content > p.
